# Session Feature Extractor (sfe)

A Python package for extracting and visualizing session-based features from data. 

## Features
- Modular design: core, utils, data, vis, defs
- CLI support via Typer
- Logging with Loguru
- Data processing with Pandas, Numpy, OpenCV, Scapy
- Model/data validation with Pydantic

## Installation
```bash
pip install .[all]
```

## Requirements
- Python 3.10+

## Usage
```python
import sfe
```

## Project Structure
- sfe/defs
- sfe/core
- sfe/utils
- sfe/data
- sfe/vis
